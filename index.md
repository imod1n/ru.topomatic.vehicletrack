# Коридор движения ТС

Плагин для расчёта траекторий движения транспортных средств и построения
коридоров движения (аналог AutoTURN) для Топоматик 360.

[Установить плагин](https://360.topomatic.ru?extensionInstallPath=https://imod1n.github.io/ru.topomatic.vehicletrack/)

[View on GitHub](https://github.com/imod1n/ru.topomatic.vehicletrack)

# Установка зависимостей
npm install

# Сборка
npm run build
